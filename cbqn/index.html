cbqn

A crate for running BQN code within a Rust program using CBQN interpreter compiled as a shared object.

Usage

Simple expressions can be run with the BQN! convenience macro. For more advanced use, the methods of BQNValue provide the necessary functionality.

Examples using the BQN! macro

let sum = BQN!("1+1");
assert_eq!(sum.into_f64(), 2.0);
assert_eq!(BQN!("‚åΩ‚â°‚ä¢", "BQN").into_f64(), 0.0);
let strs = BQN!(' ', "(‚ä¢-Àú+`√ó¬¨)‚àò=‚äî‚ä¢", "Rust ‚ù§Ô∏è BQN")
    .into_bqnvalue_vec()
    .into_iter()
    .map(BQNValue::into_string)
    .collect::<Vec<String>>();

let strings = ["join", "these", "please"];
assert_eq!(BQN!("‚àæ", strings).into_string(), "jointheseplease");
